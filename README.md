## ğŸ“Œ Project Overview

**VyaparSaathi** is a friendly, easy-to-use **Tax & Compliance Copilot** designed for micro-businesses, freelancers, and gig workers in India.

Instead of acting as a complex tax filing system, TaxSaathi works as a **guidance and awareness assistant** that:
- Explains tax concepts in simple language
- Provides proactive compliance reminders
- Answers user queries using chat and voice
- Reduces fear and confusion around tax laws

The project focuses on **confidence-building**, not legal decision-making.
